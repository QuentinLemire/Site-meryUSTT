# 🏓 Mery US TT

Site web dynamique développé pour un club de tennis de table.

## 🎯 Objectif du projet
Créer une plateforme complète permettant :
- la gestion des utilisateurs
- la gestion des équipes et des rencontres
- l’affichage des actualités et des événements
- une boutique en ligne avec panier
